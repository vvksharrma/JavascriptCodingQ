## Funcitons

#### Regular Funcitons
```
function add(x:number,y:number):number{
    return x+y;
}
```

#### Funciton expression
```
let add=function(x:number,y:number):number{
    return x+y;
}
```

#### Arrow Funciton 
```
let add=(x:number,y:number):number{
    return x+y;
}
```

#### Rest paramss
function(...numbers:Array<number>):number{
    return numbers;
}
```
